# إدارة الديلفري - تأمينات وبضاعة وطلبات

تطبيق ويب بسيط لإدارة:
- **التأمينات**: المبالغ اللي بتسيبها عند المكتب (تأمين)
- **المنتجات**: البضاعة اللي بتاخدها (مع الوحدة والكمية الحالية)
- **حركات المخزون**: إدخال بضاعة / إخراج بضاعة مع التاريخ والملاحظات
- **الطلبات**: الطلبات اللي بتطلع منها، مع الحالة (قيد الانتظار، متسلم، مستبعد، متأجل) وملاحظات

البيانات كلها بتتحفظ في **localStorage** في المتصفح مؤقتاً، لحد ما تعمل ربط بقاعدة بيانات.

## الدخول

- **كلمة المرور الافتراضية**: `delivery123`
- تغيير كلمة المرور: عدّل الملف `js/auth.js` وغيّر قيمة `APP_PASSWORD`.

## التشغيل

افتح الملف `index.html` في المتصفح (مثلاً دبل كليك عليه)، أو شغّل سيرفر محلي:

```bash
npx serve .
```

ثم افتح الرابط اللي يظهر (مثلاً http://localhost:3000).

## التقارير

من صفحة **التقارير** تقدر تصدّر:
- التأمينات → Excel أو PDF
- المنتجات والمخزون → Excel أو PDF
- حركات المخزون → Excel أو PDF
- الطلبات → Excel أو PDF

## ملاحظات

- لما تحط حالة الطلب **متسلم** يتم خصم الكميات من المخزون تلقائياً.
- الحركات كلها مسجلة بالتاريخ والملاحظات.
- في كل شاشة في إمكانية إضافة ملاحظات حسب احتياجك.
